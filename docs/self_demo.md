
# Vintic 셀프 데모 가이드 

---

### 01. 상품 이미지 업로드

* 홈 화면에서 **[상품 등록하기]** 버튼을 누르고 신발 사진을 올리는 단계예요.
* 사진은 앞면, 뒷면, 측면, 하자 부위(선택)까지 최소 3장을 세트로 올려야 합니다.
* **참고:** 사진 한 장의 용량이 50MB보다 크다면 용량을 줄여서 올려주세요.

### 02. AI 상품 분석

* 사진을 다 올리면 AI가 분석을 시작해요.
* 사진만 보고 **브랜드, 모델명, 컬러, 사이즈, 신발 등급**을 자동으로 빈칸을 채워줍니다.
* **참고:** 혹시 AI가 실수한 부분이 있다면, 유저가 직접 칸을 눌러서 올바르게 고칠 수 있습니다.

### 03. 추가 정보 입력

* 신발의 **구성품 여부**를 선택하고, AI가 쓴 설명 외에 내가 더 쓰고 싶은 말을 자유롭게 적어줍니다.
* 다 적었다면 **[AI 기준가 분석하기]** 버튼을 눌러주세요.

### 04. 판매 가격 설정

* 가격 설정 화면으로 넘어가면, AI가 적정 가격 범위와 기준가를 제안해줍니다.
* 가격 수정하기 버튼을 눌러서 범위 안에서 판매 가격을 변경할 수 있습니다.
* **참고:** "왜 이 가격이야?" 궁금하다면 [가격 산정 근거 자세히보기]를 눌러서 이유를 볼 수 있습니다.

### 05. 상품 정보 확인

* 마지막으로 내가 쓴 정보가 다 맞는지 마지막으로 보여주는 확인 화면이 나옵니다.
* 상품 등록하기 버튼을 누르면 **상품 등록 성공!**
