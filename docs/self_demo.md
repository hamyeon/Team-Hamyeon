# Vintic 셀프 데모 가이드

[https://team-hamyeon.vercel.app](https://team-hamyeon.vercel.app)

Vintic MVP의 상품 등록 흐름을 직접 시연하기 위한 가이드입니다.

데모의 핵심은 화면 전환 자체가 아니라,

**상품 이미지 업로드 → AI 상품 분석 → 시세 기반 가격 계산 → 최종 상품 등록** 흐름이 하나의 파이프라인으로 연결되는 것을 보여주는 것입니다.

<br />

## 01. 상품 이미지 업로드

홈 화면에서 **[상품 등록하기]** 버튼을 클릭하고 신발 사진을 업로드합니다.

사진은 최소 3장 이상 업로드하는 것을 권장합니다.

- 앞면
- 뒷면
- 측면
- 하자 부위 또는 오염 부위

**시연 포인트**

업로드한 이미지가 이후 AI 분석과 상품 등록 데이터에 사용된다는 점을 설명합니다.

**주의사항**

- 사진 한 장의 용량이 50MB보다 크면 업로드 전에 용량을 줄입니다.
- 너무 어둡거나 흐린 사진은 AI 분석 정확도가 떨어질 수 있습니다.

<br />

## 02. AI 상품 분석

이미지를 업로드하면 AI가 상품 정보를 분석합니다.

AI는 이미지 기반으로 다음 정보를 자동 추출합니다.

- 브랜드
- 모델명
- 컬러
- 사이즈
- 상품 상태
- 상태 등급

**시연 포인트**

AI가 가격을 직접 결정하는 것이 아니라,

사진 속 비정형 정보를 가격 계산에 사용할 수 있는 상품 데이터로 변환한다는 점을 설명합니다.

**주의사항**

AI 분석 결과가 실제 상품과 다를 수 있으므로, 분석 결과는 최종값이 아니라 사용자가 확인해야 하는 초안입니다.

<br />

## 03. AI 분석 결과 확인 및 추가 정보 입력

AI가 채운 상품 정보를 확인합니다.

분석 결과가 실제 상품과 다르다면 사용자가 직접 수정합니다.

이후 구성품 여부를 선택하고, 판매자가 추가로 설명하고 싶은 내용을 입력합니다.

예시 입력 정보는 다음과 같습니다.

- 박스 포함 여부
- 여분 끈 포함 여부
- 오염 또는 하자 설명
- 착용감
- 판매자 추가 설명

입력이 끝나면 **[AI 기준가 분석하기]** 또는 **[추천 가격 계산하기]** 버튼을 클릭합니다.

**시연 포인트**

AI 결과를 그대로 확정하지 않고, 사용자가 직접 확인하고 수정할 수 있다는 점을 보여줍니다.

<br />

## 04. 추천 가격 확인 및 판매 가격 설정

가격 설정 화면에서 시스템이 추천 가격 정보를 제공합니다.

확인할 내용은 다음과 같습니다.

- 추천 판매가
- 권장 가격 범위
- 가격 산정 이유

판매자는 추천 가격을 참고하여 최종 판매 가격을 설정합니다.

가격을 수정하려면 **[가격 수정하기]** 버튼을 클릭합니다.

**시연 포인트**

추천 가격은 AI의 단순 예측값이 아니라,

KREAM/eBay 유사 거래 시세와 상품 상태, 구성품 여부를 반영해 계산된 결과라는 점을 설명합니다.

**추가 확인**

가격 산정 근거가 궁금한 경우 **[가격 산정 근거 자세히보기]**를 클릭해 계산 이유를 확인합니다.

<br />

## 05. 상품 정보 최종 확인

최종 등록 전 상품 정보를 다시 확인합니다.

확인할 항목은 다음과 같습니다.

- 상품 이미지
- 브랜드 / 모델명 / 컬러 / 사이즈
- 상태 등급
- 구성품 여부
- 판매자 설명
- 추천 가격
- 최종 판매 가격

정보가 맞다면 **[상품 등록하기]** 버튼을 클릭합니다.

<br />

## 06. 상품 등록 완료 및 리스트 확인

상품 등록이 완료되면 상품 등록 성공 화면을 확인합니다.

이후 상품 찾기 또는 상품 리스트 페이지에서 방금 등록한 상품이 조회되는지 확인합니다.

**시연 포인트**

최종적으로 상품 이미지, AI 분석 정보, 추천 가격, 최종 판매 가격이 하나의 상품 데이터로 저장되고 조회되는 것을 보여줍니다.

<br />

## 데모 성공 기준

데모가 정상적으로 완료되었다면 다음 흐름이 모두 확인되어야 합니다.

1. 상품 이미지가 업로드된다.
2. AI가 상품 정보를 분석한다.
3. 사용자가 AI 분석 결과를 확인하고 수정할 수 있다.
4. 구성품 여부와 추가 설명을 입력할 수 있다.
5. 추천 판매가와 권장 가격 범위가 표시된다.
6. 최종 판매 가격을 설정할 수 있다.
7. 상품이 등록된다.
8. 등록된 상품이 상품 리스트에서 조회된다.

<br />

## 데모 중 문제가 발생할 경우

### 이미지 업로드 실패

- 이미지 용량이 너무 큰지 확인합니다.
- 지원하지 않는 파일 형식인지 확인합니다.
- 네트워크 연결 상태를 확인합니다.

### AI 분석 결과가 부정확한 경우

- 사용자가 직접 브랜드, 모델명, 색상, 상태 정보를 수정합니다.
- 이 상황은 실패가 아니라, AI 결과를 사용자 확인으로 보정하는 흐름을 보여줄 수 있는 사례입니다.

### 추천 가격 계산이 실패한 경우

- 구성품 여부와 필수 상품 정보가 입력되었는지 확인합니다.
- 유사 거래 데이터가 부족한 경우 가격 계산 결과가 제한될 수 있습니다.

### 상품 등록이 실패한 경우

- 최종 판매 가격이 입력되었는지 확인합니다.
- 필수 상품 정보가 비어 있는지 확인합니다.
